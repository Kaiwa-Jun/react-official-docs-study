# React Official Docs Workshop

React公式ドキュメント（https://ja.react.dev/）を教材にした、
少人数・長期スパンのReact勉強会用リポジトリです。

このリポジトリには、各回で使用するデモコードやサンプルを格納します。
参加者全員がローカルで動かしながら学べることを前提としています。

---

## 勉強会の目的・立ち位置

- Reactを「書ける」だけでなく「なぜそう書くのか」を理解する
- AIコーディングに頼りすぎず、本質的なReactの考え方を身につける
- React公式ドキュメントを自力で読めるようになる土台を作る

---

## 開催概要

- 開催頻度：毎週土曜 13:00〜
- 時間：1〜2時間（テーマによって短縮可）
- テーマ数：毎回1テーマのみ

---

## 進め方

- React公式ドキュメントを主教材とする
- 毎回1テーマに絞って深掘りする
- 勉強会のために事前調査・準備を行い、場ではアウトプットを中心に進める
- デモ・サンプルは必ずブラウザで動く状態で用意する
- 公式ドキュメントを読みつつ、コードと画面を行き来しながら進める
- 作成したデモ・サンプルコードはGitHubで管理する
- 参加者はローカル環境で同じコード・表示を再現できることを前提とする

---

## 勉強会のルール

- 気になったことは遠慮せず質問する
- 分かる範囲で回答し、分からないことはその場で調べる
- 両者とも「分からない状態」を許容する
- 正解を急がず、理解を優先する

---

## 公式ドキュメントの扱い方

- 基本方針として `/learn` セクションを中心に進める
- 概念理解 → 基本構文 → 考え方、の順で扱う
- 理解が進んだ段階で `/reference` をテーマ単位で取り上げる
- ドキュメントの構成順＝勉強会の進行順ではない

---

## ディレクトリ構成（例）


/01-quick-start
/02-describing-ui
/03-adding-interactivity
...

各ディレクトリには、その回のデモコードと簡単な補足READMEを配置します。
